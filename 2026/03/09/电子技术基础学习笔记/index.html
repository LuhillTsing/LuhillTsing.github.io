第一部分　模拟电子技术

第1章　半导体器件

1.1　半导体基础知识

本征半导体是纯净的晶体硅或锗，其导电能力介于导体和绝缘体之间。在本征半导体中，热激发会产生电子-空穴对，载流子浓度受温度影响。当温度升高时，载流子浓度增加，导电能力增强。

P型半导体是在本征半导体中掺入三价元素（如硼），空穴为多数载流子；N型半导体是在本征半导体中掺入五价元素（如磷），电子为多数载流子。PN结是P型半导体和N型半导体结合形成的空间电荷区，具有单向导电性。

1.2　二极管

二极管由PN结构成，具有单向导电性。正向偏置时，二极管导通，正向电流随正向电压增加而快速增加；反向偏置时，二极管截止，反向电流很小。

  • 稳压二极管：工作在反向击穿区，在反向电压作用下，稳压管两端电压基本保持不变，可用于稳定电压
  • 发光二极管（LED）：正向导通时会发出可见光
  • 光电二极管：将光信号转换为电信号的器件

1.3　晶体管

**晶体管（Transistor）**是由两个PN结组成的三端器件，有NPN和PNP两种类型。晶体管具有电流放大作用，是放大电路的核心器件。

共射极放大电路是最常用的基本放大电路，输出信号与输入信号相位相反，电压增益较高。晶体管工作在放大区时，发射结正偏，集电结反偏。

主要参数包括：

  • 电流放大系数 β
  • 穿透电流 I_CEO
  • 反向截止电流 I_CBO

1.4　场效应管

**场效应管（FET）**是一种电压控制型器件，输入电阻极高，可达10^9Ω以上。

  • 结型场效应管（JFET）：利用栅极-沟道间的PN结反向偏置来控制漏极电流
  • 绝缘栅型场效应管（MOSFET）：利用栅极与沟道间的SiO2绝缘层进行电场效应控制，分为N沟道和P沟道增强型、耗尽型四种类型

场效应管具有噪声低、功耗小、输入电阻高等优点，广泛应用于模拟和数字电路中。

第2章　基本放大电路

2.1　放大电路的基本概念

放大电路的核心功能是将微弱的电信号放大到所需的幅度。主要性能指标包括：

  • 放大倍数（增益）
  • 输入电阻
  • 输出电阻
  • 通频带
  • 失真度
  • 噪声系数

按照工作状态可分为：

  • 甲类（A类）：整个周期内均有电流流过晶体管，效率低但失真小
  • 乙类（B类）：只有半个周期导通，效率高但存在交越失真
  • 甲乙类（AB类）：效率略低但失真较小
  • 丙类（C类）：效率高但失真大

2.2　共射极放大电路

共射极放大电路是最基本的单管放大电路，晶体管的发射极作为输入和输出回路的公共端。该电路具有较高的电压增益和电流增益，但输入电阻适中，输出电阻较高。

静态工作点设置对放大电路的性能影响很大：

  • 工作点过高 → 饱和失真
  • 工作点过低 → 截止失真

稳定静态工作点的常用方法：射极偏置电路、负反馈、温度补偿等。

2.3　共集电极和共基极放大电路

  • 共集电极放大电路（射极输出器）：输出取自发射极，电压增益略小于1，但具有高输入电阻和低输出电阻的特点，常用作阻抗变换器或缓冲级
  • 共基极放大电路：输入端在发射极，输出端在集电极，具有低输入电阻和高输出电阻的特点，频率特性好，常用于高频放大电路

2.4　多级放大电路

多级放大电路由多个单级放大电路级联组成，以获得更高的总增益和更宽的通频带。

级间耦合方式：

  • 直接耦合：可以放大直流信号和缓慢变化信号，但存在零点漂移问题
  • 阻容耦合：只能放大交流信号，但各级静态工作点相互独立
  • 变压器耦合
  • 光电耦合

第3章　集成运算放大电路

3.1　集成运算放大器概述

集成运算放大器是一种高增益的直接耦合多级放大电路，内部通常包含输入级、中间级、输出级和偏置电路。

主要特点：

  • 高开环增益
  • 高输入电阻
  • 低输出电阻
  • 共模抑制比高
  • 失调电压小

常用参数：开环电压增益 Avo、输入失调电压 V_IO、输入偏置电流 I_IB、转换速率 SR、单位增益带宽 GB

理想运算放大器：具有无限大的开环增益、输入电阻和带宽，零输出电阻和失调电压。

3.2　基本运算电路

  • 反相放大电路：输出信号与输入信号相位相反，输入端为虚地，电压增益由反馈电阻与输入电阻之比决定
  • 同相放大电路：输出信号与输入信号相位相同，电压增益大于1，输入电阻极高
  • 差分放大电路：能够放大两个输入信号的差值，抑制共模信号
  • 积分电路和微分电路：模拟计算机的基本单元
  • 对数放大器和指数放大器：利用晶体管的指数特性实现

3.3　有源滤波电路

有源滤波器由运算放大器和RC网络组成，能够在滤波的同时提供增益。

滤波器类型：

  • 低通滤波器（LPF）
  • 高通滤波器（HPF）
  • 带通滤波器（BPF）
  • 带阻滤波器（BEF）

滤波器特性：

  • 一阶滤波器：-20dB/十倍频程
  • 二阶滤波器：-40dB/十倍频程
  • 巴特沃斯滤波器：平坦的通带幅频特性
  • 切比雪夫滤波器：通带内有纹波但过渡带更陡
  • 贝塞尔滤波器：线性相位特性

第4章　反馈放大电路

4.1　反馈的基本概念

反馈是将放大电路输出信号的一部分或全部送回输入端，与输入信号进行比较的过程。

  • 正反馈：增强输入信号
  • 负反馈：削弱输入信号

负反馈虽然降低了放大倍数，但能够：

  • 提高放大倍数的稳定性
  • 减小非线性失真
  • 扩展通频带
  • 改变输入输出电阻

反馈类型：

  • 按取样方式：电压反馈、电流反馈
  • 按比较方式：串联反馈、并联反馈

反馈深度：F = 1 + Aβ

4.2　负反馈放大电路的分析

  • 电压串联负反馈：提高输入电阻、降低输出电阻
  • 电压并联负反馈：降低输入电阻和输出电阻
  • 电流串联负反馈：提高输入电阻和输出电阻
  • 电流并联负反馈：降低输入电阻、提高输出电阻

深度负反馈条件下：闭环增益 A_f ≈ 1/β，基本取决于反馈网络

4.3　负反馈放大电路的自激振荡

产生条件：环路增益 Aβ = -1

  • 幅度条件：|Aβ| = 1
  • 相位条件：φ_A + φ_β = ±180°

消除方法：

  • 补偿技术（主极点补偿、滞后补偿、超前-滞后补偿）
  • 调整反馈网络参数
  • 加入消振电容或RC消振网络

第5章　功率放大电路

5.1　功率放大电路的特点与分类

功率放大电路的主要任务是在允许失真范围内输出足够的功率驱动负载。

  • 乙类推挽功率放大器效率可达 78.5%
  • 甲乙类放大器效率略低但失真较小

注意事项：

  • 最大集电极功耗 P_CM
  • 最大集电极-发射极电压 V_CEO
  • 最大集电极电流 I_CM
  • 散热设计

5.2　OCL和OTL电路

  • OCL电路：双电源互补对称功率放大电路，采用NPN和PNP晶体管互补推挽工作，需要正负双电源供电
  • OTL电路：单电源互补对称功率放大电路，输出端采用大电容耦合

集成功放器件：将功率放大电路集成化，内部包含前置放大、功率放大、过流保护、过热保护等功能

第6章　振荡电路

6.1　正弦波振荡电路的基本原理

正弦波振荡电路是一种不需要输入信号就能自动产生周期性正弦波输出的电路。

振荡的平衡条件：Aβ = 1

  • 幅度平衡条件：|Aβ| = 1
  • 相位平衡条件：φ_A + φ_β = 2nπ

起振条件：|Aβ| > 1

6.2　LC振荡电路

LC振荡电路采用LC并联谐振回路作为选频网络，振荡频率由 ω = 1/√(LC) 决定。

  • 变压器耦合振荡电路
  • 电感三点式振荡电路（哈特莱振荡器）
  • 电容三点式振荡电路（科皮兹振荡器）：频率稳定性好

6.3　RC振荡电路

RC振荡电路采用RC串并联网络作为选频网络，适用于低频振荡。

RC文氏电桥振荡电路：振荡频率 f_0 = 1/(2πRC)

6.4　晶体振荡电路

晶体振荡电路利用石英晶体的压电谐振特性，具有极高的频率稳定度。

  • 串联型晶体振荡器
  • 并联型晶体振荡器（应用更广泛）
  • 晶振频率范围从kHz到MHz，是数字系统中的理想时钟源

第7章　直流稳压电源

7.1　整流电路

直流稳压电源通常由电源变压器、整流电路、滤波电路和稳压电路组成。

整流电路类型：

  • 单相半波整流：简单但输出脉动大
  • 单相全波整流：输出脉动较小
  • 单相桥式整流：最常用，变压器利用率高

7.2　滤波电路

  • 电容滤波：最常用，利用电容的储能特性
  • 电感滤波：适用于大电流负载
  • LC滤波和π型滤波：效果更好

7.3　稳压电路

  • 串联型稳压电路：最常用的稳压方式
  • 集成稳压器：78系列（固定正电压）、79系列（固定负电压）、LM317/LM337（可调输出）
  • 开关稳压电路：效率高但电路复杂

第二部分　数字电子技术

第1章　数字逻辑基础

1.1　数制与编码

数字电路中常用二进制、八进制、十六进制表示数值。

  • BCD码（8421码）：用4位二进制表示一位十进制数字
  • 常用编码：8421码、2421码、余3码、格雷码
  • ASCII码：用7位二进制表示128个字符

1.2　逻辑代数基础

基本逻辑运算：与（AND）、或（OR）、非（NOT）

复合逻辑运算：与非、或非、与或非、异或、同或

摩根定理（反演定律）：

  • $\overline{A+B} = \overline{A} \cdot \overline{B}$
  • $\overline{A \cdot B} = \overline{A} + \overline{B}$

1.3　逻辑函数的化简

  • 代数化简法：利用逻辑代数定律和公式
  • 卡诺图化简法：图形化简方法，相邻格子合并
  • 约束条件：实际电路中不可能出现的变量组合

第2章　组合逻辑电路

2.1　组合逻辑电路的分析与设计

分析步骤：写出逻辑表达式 → 化简 → 列出真值表 → 说明电路功能

设计步骤：确定输入输出变量 → 列出真值表 → 写出逻辑表达式 → 化简 → 画逻辑图

竞争-冒险：门电路延迟造成，可能产生尖峰脉冲

消除方法：加选通脉冲、加滤波电容、增加冗余项

2.2　常用组合逻辑电路

  • 编码器：多个输入信号转换为二进制代码（74LS148优先编码器）
  • 译码器：二进制代码转换为对应输出（74LS138、74LS48）
  • 数据选择器（MUX）：多选一（74LS151）
  • 数据分配器（DMUX）：一选多
  • 数值比较器：比较二进制数大小（74LS85）
  • 全加器：一位二进制数相加

2.3　组合可编程逻辑器件

  • PAL：可编程与阵列、固定或阵列
  • GAL：可编程输出逻辑宏单元（OLMC）
  • PLA：与阵列和或阵列均可编程
  • CPLD：复杂可编程逻辑器件
  • FPGA：现场可编程门阵列（基于查找表结构）

第3章　时序逻辑电路

3.1　触发器

触发器是时序逻辑电路的基本单元，具有记忆功能，能够存储1位二进制信息。

  • RS触发器：基本触发器，有复位和置位功能
  • JK触发器：功能完善，J=K=1时状态翻转
  • D触发器：时钟上升沿将D输入传递到输出（最常用）
  • T触发器：每来一个时钟脉冲状态翻转一次

3.2　寄存器

寄存器用于存储一组二进制数据。

  • 基本寄存器：并行输入并行输出
  • 移位寄存器：可左移或右移
  • 串入并出、并入串出移位寄存器

3.3　计数器

计数器是统计输入脉冲个数的时序电路。

  • 异步计数器（行波计数器）：串行连接，速度较慢
  • 同步计数器：同一时钟驱动，速度快
  • 模n计数器：计数循环长度为n
  • 集成计数器：74LS160、74LS161、74LS290

3.4　时序逻辑电路的分析与设计

分析步骤：写时钟方程、驱动方程、输出方程 → 代入特性方程 → 求状态方程 → 列出状态转换表 → 画状态图和时序图

设计步骤：建立状态图/状态表 → 状态化简 → 状态编码 → 选择触发器类型 → 求驱动方程和输出方程 → 画逻辑图

第4章　半导体存储器

4.1　只读存储器（ROM）

  • 掩模ROM：制造时固定存储内容
  • PROM：一次性编程
  • EPROM：紫外线擦除
  • EEPROM、Flash：电信号擦除

4.2　随机存取存储器（RAM）

  • SRAM：触发器构成，存取速度快，用于高速缓存
  • DRAM：电容存储电荷，需要周期性刷新，用于主存储器

4.3　存储器容量的扩展

  • 字扩展：增加存储单元数量
  • 位扩展：增加每个存储单元的位数

第5章　脉冲产生与变换

5.1　多谐振荡器

多谐振荡器是不需要外加触发信号就能自动产生矩形波的自激振荡电路。

  • TTL门电路构成
  • 施密特触发器构成
  • 石英晶体多谐振荡器（频率稳定度极高）

5.2　单稳态触发器

只有一个稳定状态，触发后进入暂稳态，然后自动返回。

  • 微分型单稳态触发器
  • 积分型单稳态触发器
  • 集成单稳态触发器：74LS121、74LS123

5.3　施密特触发器

具有两个阈值电压（VT+和VT-），具有回差电压ΔVT。

  • 将任意波形整形为矩形波
  • 常用芯片：74LS14、74LS132

5.4　555定时器

多用途集成定时器芯片，内部包含比较器、RS触发器和放电晶体管。

  • 可构成多谐振荡器、单稳态触发器、施密特触发器
  • 工作电源范围宽：4.5V-16V
  • 输出电流大：200mA

第6章　数模与模数转换

6.1　数模转换器（D/A转换器）

将数字量转换为模拟量。

  • 权电阻网络D/A转换器
  • T型电阻网络D/A转换器（最常用）

主要参数：分辨率、转换精度、转换速度、线性度

6.2　模数转换器（A/D转换器）

将模拟量转换为数字量。

  • 并联比较型：速度最快
  • 逐次逼近型：最常用
  • 双积分型：抗干扰能力强，适合低速高精度
  • Σ-Δ型：分辨率高，用于高精度音频

6.3　采样保持电路

  • 采样期间跟踪输入信号
  • 保持期间保持输入稳定
  • 奈奎斯特采样定理：采样频率 ≥ 2 × 输入信号最高频率

幅度 连续 离散
