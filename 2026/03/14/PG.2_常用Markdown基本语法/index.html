一、模板概述

1.1 模板介绍

本文档是一份通用 Hexo + NexT 主题博客文章模板，旨在为博客作者提供一个功能完备、样式丰富的写作范本。通过从本文档复制内容到您的 _posts/ 目录下的 .md 文件中，您可以快速创建具有专业外观和丰富功能的博客文章。

本模板基于 Hexo 静态网站生成器 和 NexT 主题 构建，完美兼容 Gemini 或 Pisces 布局方案。NexT 主题是 Hexo 生态中最受欢迎的主题之一，以其优雅的设计和强大的功能著称。

1.2 核心特性

本模板为您提供了以下核心功能支持：

  • 内容加密：保护敏感文章内容，支持密码访问
  • 目录导航：自动生成文章目录，支持层级折叠
  • 标签管理：灵活的标签和分类系统
  • 版权声明：多种版权许可证选项
  • AI声明：规范的AI创作内容标识
  • 数学公式：MathJax 和 KaTeX 支持
  • 流程图：Mermaid 和 Flowchart 支持
  • 多媒体：音频和视频嵌入

三、基础 Markdown 语法

3.1 文本格式

这是一段普通的段落，用于展示基础的 Markdown 语法。您可以加粗文字，也可以斜体文字，或者进行删除线测试。

加粗文本的使用场景包括：

  • 强调重要概念
  • 标题和关键术语
  • 行动号召

斜体文本通常用于：

  • 引用和说明
  • 书籍、电影等作品名称
  • 专业术语

删除线用于表示：

  • 过时信息
  • 错误内容
  • 待删除的文本
