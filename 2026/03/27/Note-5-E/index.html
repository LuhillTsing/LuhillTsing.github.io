第二模块：动词系统

2.1 动词分类

类型 说明 示例
行为动词 及物/不及物 eat, run
连系动词 be/感官/变化 is, look, become
助动词 帮助构成时态/语态 be, do, have, will
情态动词 表示能力/许可/推测 can, must

2.2 情态动词速查表

情态动词 能力 许可 推测 义务
can ✓ ✓ 可能 -
could ✓(过去) ✓(委婉) 较小可能 -
may - ✓ 可能 -
might - ✓(更委婉) 较小可能 -
must - - 一定 必须
have to - - - 必须(客观)
should - - 应该 应该
will - - 将要 -

2.3 情态动词+have done

形式 含义 例句
must have done 一定做过 He must have left.
may have done 可能做过 He may have missed it.
can’t have done 不可能做过 He can’t have done that.
should have done 应该做过(却没) You should have tried.
needn’t have done 不必做过(却做了) You needn’t have rushed.

第三模块：时态系统

3.1 十六种时态矩阵

3.2 常用时态核心对比

时态 结构 用法 时间状语
一般现在 do/does 习惯/真理/特征 always, usually
一般过去 did 过去动作/状态 yesterday, ago
一般将来 will do 将来动作/决定 tomorrow, soon
现在进行 am/is/are doing 正在进行 now, right now
过去进行 was/were doing 过去某时进行 at that time
现在完成 have/has done 已完成/影响至今 already, yet, since
过去完成 had done 过去的过去 before, after

3.3 易错时态辨析

易错点 错误 正确 说明
have gone vs have been I have gone to Beijing I have been to Beijing gone=去了(不在), been=去过
瞬间动词+时间段 He has died for 3 years He died 3 years ago / He has been dead for 3 years 瞬间动词不与时间段连用
短暂动作接段时间 I have come here for 2 weeks I have been here for 2 weeks come→be

4.1 被动语态构成

4.2 各时态被动语态

时态 主动 被动
一般现在 do am/is/are done
一般过去 did was/were done
一般将来 will do will be done
现在完成 have done have/has been done

第五模块：非谓语动词

5.1 三种非谓语对比

类型 形式 主/动特征 时间特征
不定式 to do 未发生/将来 将来
动名词 doing 已发生 进行
分词 doing(现在)/done(过去) 主动/被动 进行/完成

5.2 不定式用法速查

用法 结构 示例
作主语 To do is… To learn English is important.
作宾语 want/ decide/ hope to do I want to go.
作表语 be + to do My goal is to become a doctor.
作定语 名词 + to do I have work to do.
作状语 表目的/结果 I came here to study.
宾补 ask/tell sb to do I asked him to help.

5.3 感官/使役动词后宾补

动词类型 + 宾语 + 原形 + 宾语 + doing
感官动词 see/watch/hear/notice + O + do 表示动作完成
使役动词 make/let/have + O + do -
get - get + O + doing

5.4 动名词与不定式选择

接动名词 接不定式 意义不同
enjoy, finish want, hope remember doing(已做) vs remember to do(要做)
mind, avoid decide, agree stop doing(停止) vs stop to do(停下去做)
suggest, practice expect, promise try doing(尝试) vs try to do(努力)

7.4 that与which选择

用that 用which
先行词为all/everything/nothing等 非限制性定语从句
先行词被最高级/序数词修饰 介词后
先行词为人与物混合 先行词为that

附录三：语法口诀速记

A3.1 冠词

  • 元音字母(a,e,i,o,u)开头 → an
  • 辅音字母开头 → a
  • 特例：an hour, an honest man / a university, a European

A3.2 时间介词

  • 年月周前要用 in
  • 日子前面该用 on
  • 时刻点前要用 at

A3.3 主谓一致

  • 就近原则：either…or, neither…nor, not only…but also
  • 意义一致：family/team 整体单数/成员复数

A3.4 感官动词

  • 一感：feel
  • 二听：listen to, hear
  • 三观看：watch, see, notice, observe
  • +宾+do（表动作全过程）/ +宾+doing（表动作进行）
