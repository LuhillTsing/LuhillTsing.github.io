货币基金的选择

发表于 2026-02-26 更新于 2026-03-01 分类于 finances

一、货币基金

货币基金全称货币市场基金，是由基金管理人运作、基金托管人保管资金的开放式基金，主要投资范围为具有良好流动性的金融工具，包括现金，期限在一年以内（含一年）的银行存款、债券回购、中央银行票据、同业存单，剩余期限在397天以内（含397天）的债券、非金融企业债务融资工具、资产支持证券等其他具有良好流动性的货币市场工具。具有高安全性、流动性强、投资门槛低和投资成本低的特点。其收益通常高于活期存款，一般以中国人民银行公布的七天通知存款利率（税后）为业绩比较基准，通常被视为现金管理工具。

二、货币基金A、B份额的区别

份额类别 主要面向对象 典型申购门槛 典型销售服务费率（年化）
A类 个人投资者 极低（如1分、1元、100元） 较高（如0.25%）
B类 机构投资者 非常高（通常为500万元起） 较低（如0.01%）

货币基金A类份额和B类份额的主要区别是申购门槛和销售服务费率。
货币基金B类份额以前是面向机构投资者的，投资门槛较高，所以费率较低。现在个人投资者也可以购买货币基金的B类份额了，因为它的申购门槛变低了，但费率还是和以前一样低。
除销售服务费率以外，基金运作还有管理费、托管费，较低的管理费、托管费可以节省投资成本以提升投资回报收益。管理费、托管费、销售服务费每个交易日在基金净值中扣除，因此对部分投资者的感知比较低，但以上费用可能会吞噬一部分投资收益。

三、资金规模

规模区间 优点 缺点
过小（<10亿元） - 流动性风险高：大额赎回易冲击运作
清盘风险：若连续60日低于5000万可能清盘
收益偏低：议价能力弱
适中（约50亿元-500亿元） 收益潜力佳：有议价能力，调仓灵活
稳健性好：能较好兼顾安全与收益		 需要从同规模产品中筛选
超大（>500亿元） 安全性高：抗赎回冲击能力强
流动性好：应对日常申赎能力强		 收益可能钝化：资金量过大难以全部配置高收益资产
灵活性低：调仓慢

五、申购和赎回

5.1 快速赎回

货币基金分为支持T+0快速赎回（宝类基金，这类基金名称中通常带有“宝”字）和T+1常规赎回（非宝类基金，这类基金名称中通常不含“宝”字）。支持T+0快速赎回的基金通常可以在发起赎回后当天到账，T+1常规赎回的基金通常在发起赎回后下一个交易日到账。

需要注意的是，大多数支持T+0快速赎回的货币基金有快速赎回额度限制，通常为10千元。如有大额资金需要使用快速赎回功能，可以分散投资在多支支持T+0快速赎回功能的货币基金中。

5.2 基金申购规则

货币基金的申购和赎回适用基金一般交易规则。在当天交易收盘截止时刻前申购的基金份额按照当天收盘基金净值成交，在下一个交易日确认基金份额并计算收益。在上一个交易日收盘截止时刻后申购的基金份额，按照下一个交易日收盘截止时刻基金净值成交，在申购后的第二个交易日确认基金份额并计算收益。

5.3 基金赎回规则

货币基金的赎回在交易日收盘截止时刻前申请赎回的基金份额，按照当天收盘截止时刻基金净值成交，计算当天收益，在下一个交易日确认赎回份额并到账赎回资金。在交易日收盘截止时刻后申请赎回的基金份额，按照下一个交易日收盘截止时刻基金净值成交，收益计算至收盘截止时刻，在申购后的第二个交易日确认基金份额并到账赎回资金。

支持T+0快速赎回的货币基金，按照上一个交易日收盘截止时刻的基金净值计算收益，并于发起赎回的当天到账赎回资金，赎回资金由垫付银行垫付，上一个交易日收盘截止时刻至下一个交易日收盘截止时刻期间所产生的收益归垫付银行所有。

5.4 节假日前一个交易日不赎回、节假日期间不申购

因此，如果在节假日前一个交易日申请赎回，将按照节假日后前一个交易日收盘截止时刻基金净值成交，于节假日后第一个交易日到账收益，将损失节假日期间的收益。
如果在节假日期间及节假日前一个交易日收盘时刻后申购，休市期间不计算收益，将按照下一个交易日收盘时刻基金净值成交，并于节假日后第二个交易日确认基金份额并计算收益。

BA7KTB电台简介

发表于 2025-12-25 更新于 2026-02-28 分类于 amateur-radio

1. 简介

本台系经广东省工业和信息化厅批准设立的无线电台（业余业务）。

呼号：BA7KTB
设备RIG: 泉盛Quansheng UV-K5
天馈ANT: 原装天线 ORIGINAL
电子邮箱E-MAIL: ba7ktb@qq.com
QRZ主页: Click here

本台依据《中华人民共和国无线电管理条例》和《业余无线电台管理办法》之规定，经广东省工业和信息化厅准予行政许可，取得中华人民共和国工业和信息化部统一制作的《中华人民共和国无线电台执照》。业余无线电台呼号BA7KTB，通信使用区域为全国。
本台负责人依据《业余无线电台管理办法》取得广东省工业和信息化厅核发的B类业余无线电操作技术能力验证证书，具备工作在30MHz以下频段且最大发射功率小于15瓦，或者工作在30MHz以上频段且最大发射功率不大于25瓦的业余无线电台操作技术能力。

2. 通联确认卡

根据业余无线电通联习惯，本台设计和制作了《BA7KTB通联确认卡》作为与本台确认联络的有效凭证。

2.1 签发条件

与本台完成基本通联的业余无线电台，可以获得本台签发的通联确认卡。基本通联应当相互正确抄收对方呼号并提供信号报告。无法识别并正确抄收对方呼号、不能提供信号报告的，不能获得本台签发的通联确认卡。

2.2 卡片类型

本台支持实体通联确认卡和电子通联确认卡，两者具有同等效力。选择实体通联确认卡的，同时提供电子通联确认卡；选择电子通联确认卡的，不再提供实体通联确认卡。同一条通联日志，只能签发一次通联确认卡，遗失不补。

2.3 获取方式

通过电子邮件，发送基本通联信息，实现电子通联确认。如需要实体通联确认卡，应当提供本人呼号、姓名（如适用）、电话、联系地址等联系信息。未提供联系信息且未明确要求提供实体通联确认卡的，默认提供电子通联确认卡。

2.4 签发时效

根据《业余无线电台管理办法》第三十六条之规定，本台通联日志保留两年。通联确认卡应当在通联结束后两年内签发。
收到电子邮件后，本台将在30个工作日内核实通联情况并反馈签发结果。符合签发条件的，同时提供电子通联确认卡。本台将在180个工作日内通过邮政信函或快递包裹将实体通联确认卡寄递到您提供的联系地址。

2.5 查询

本台签发的通联确认卡支持查询，查询方式为扫描通联确认卡所附的电子通联确认卡二维码。为保障信息安全，查询通联确认卡详细信息需要提供密码，密码为贵台通联时所使用的呼号。

3. 收听确认卡

根据业余无线电通联习惯，本台设计和制作了《BA7KTB收听确认卡》作为收听本台的有效凭证。

3.1 签发条件

通过业余无线电台、对讲机、收音机等各类无线电接收设备收到本台信号并向本台提供信号报告的业余无线电台、自然人可以获得本台签发的收听确认卡。

3.2 卡片类型

本台仅提供电子收听确认卡（E-SWL），不提供实体收听确认卡。本台签发的电子收听确认卡是收听到本台的有效凭证。

3.3 获取方式

通过电子邮件，发送收听报告。收听报告应当具备本台呼号（姓名，如适用）、信号报告、收听频率、通联时间（时刻）等基本信息，还可以发送收听设备、天馈、收听地址等其它信息，鼓励发送录音作为信号报告的原始载体。

3.4 签发时效

根据《业余无线电台管理办法》第三十六条之规定，本台通联日志保留两年。收听确认卡应当在通联结束后两年内签发。
收到电子邮件后，本台将在30个工作日内核实通联情况并反馈签发结果。符合签发条件的，同时提供电子收听确认卡。

3.5 查询

本台签发的具备E-swl二维码的收听确认卡支持查询，查询方式为扫描收听确认卡所附的E-swl二维码，查询内容显示收听确认卡所载信息。为保障信息安全，查询收听确认卡详细信息需要提供密码，密码为贵台发送收听报告时所使用的呼号；如不具备呼号，密码为收听确认卡电子签名的前六位数字和字母（不含字符）。

4. 眼球卡

根据业余无线电活动习惯，本台使用《通联确认卡》作为本台的眼球卡（Eyeball QSO）。

4.1 签发条件

眼球卡是本台签发的具有纪念意义的卡片，向业余无线电爱好者（“火腿”）或未正式取得业余无线电呼号的业余无线电爱好者（“香肠”）发放。

4.2 卡片类型

眼球卡使用本台的“通联确认卡”签发，并在卡片显要位置注明“眼球卡”。在特定时候，眼球卡还可以使用本台停止签发的“通联确认卡”签发，并在卡片显要位置注明“眼球卡”。
本台提供眼实体眼球卡和电子眼球卡，但不单独提供电子眼球卡。

4.3 获取方式

在参加线下活动时，根据活动需要，由电台负责人当场签发。

4.4 查询

本台签发的具备E-qsl二维码的眼球卡支持查询，查询方式为扫描眼球卡所附的E-qsl二维码。查询内容显示“眼球卡”或眼球卡所载信息。部分停止签发的通联确认卡不具有E-qsl二维码，故不具备查询条件。
